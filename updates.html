Smart Header

The new smart header allows for the header to adapt around the logo

Assuming the text within the 'h1' tag in the header is on a single line, it will alter it's horizontal and vertical positioning based on the height and width of the logo. The text will automatically position itself vertically at half the logo's height (also taking into concideration the text size), and horizontally positioned to be the width of the page_maxWidth variable, minus the width of the logo, and centrally aligned to sit in the middle.

This feature can be enabled by adding '@include vAlign_header;' and '@include hAlign_header;' in your project's 'page_setup.scss' file. Comment out or remove these two lines of code to disable the feature.
